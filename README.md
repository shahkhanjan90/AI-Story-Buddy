# AI-Story-Buddy

A minimal React Native Expo app UI for generating kid-friendly story prompts.

## Included UI

- Age selector with 3 tiles:
  - `2-4 years`
  - `5-7 years`
  - `8-10 years`
- Character text input
- `Generate Story` button
- Story display area
- `Play Audio` button

## Run

```bash
npm install
npm start
```

Then open the app in Expo Go, an Android emulator, or an iOS simulator.

## Web

```bash
npm run web
```

This starts the Expo web build so the app can be opened in a browser.
